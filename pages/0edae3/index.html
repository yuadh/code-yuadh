# 基本概念

flexbox 是一种一维的布局模型，它给 flexbox 的子元素之间提供了强大的空间分布和对齐能力

# 两根轴线

flex 布局的 主轴 和 交叉轴

由 flex-direction 定义

  • row
  • row-reverse
  • column
  • column-reverse

笔记

row 和 row-reverse 属性是设置主轴为行方向的布局 ，row-reverse 改变了主轴的起始位置

# flex容器

声明了 flex 的父元素，其默认值

  • flex-direction 为 row
  • 子元素宽度不会扩张但是会缩小
  • 未定义子元素的高度为自动扩展适应父级大小
  • flex-basis 为 auto
  • flex-warp 为 nowarp

# flex-warp和flex-flow

flex-flow 是 warp 和 direction 的简写

flex-flow: row warp

warp的属性值

  • warp
  • nowarp
  • warp-reverse

笔记

# flex属性

  • flex-grow : 可用空间，如果允许延展会自动延展可用空间，由元素的 grow 值去分配可用空间
  • flex-shrink：收缩空间，
  • flex-basis： 元素大小

书写顺序 ：grow - shrink - basis
